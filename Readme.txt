# Face Detection Using MATLAB

This project detects human faces in images using MATLAB's Computer Vision Toolbox.

## How to Run
1. Open `facedetection.m` in MATLAB.
2. Make sure `VK and MSD.png` is in the same folder.
3. Run the script.

## Output
Annotated image showing detected faces.

## Tools
- MATLAB
- Computer Vision Toolbox
